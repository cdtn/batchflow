Xception¶

François Chollet. “Xception: Deep Learning with Depthwise Separable Convolutions”

class Xception(*args, **kwargs)[source]¶

Bases: batchflow.models.tf.base.TFModel

Xception model architecture.

Parameters

  • inputs (dict) – Dictionary with ‘images’ (see _make_inputs()).

  • body (dict) –

    entry, middle, exit : dict Dictionary with parameters for entry encoding: downsampling of the inputs.

    num_stagesint

    Number of block’s in the respective flow.

    filterslist of sequences of 3 ints

    Number of filters inside for individual block.

    stridesint or list of ints

    Stride of the middle separable_block inside block.

    depth_activationbool or list of bools

    Whether to use activation between depthwise and pointwise convolutions.

    combine_op{‘sum’, ‘softsum’}

    Whether to use convolution for skip-connections inside separable_block or just sum skip and output.
